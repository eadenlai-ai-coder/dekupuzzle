# Deku复原大挑战

10关拼图游戏，帮助deku复原图片！

## 文件结构

```
puzzle-game/
├── index.html          # 游戏主文件
├── README.md           # 本说明文件
└── images/             # 图片文件夹
    ├── 1.jpg
    ├── 2.jpg
    ├── ... (10+张图片)
    └── 10.jpg
```

## 游戏规则

- **第1-4关**：2×2 开门
- **第5-8关**：3×3 人上人
- **第9关**：4×4 顶级
- **第10关**：5×5 夯

每关随机抽取一张未使用过的图片，通关后自动解锁下一关。

## 操作说明

- 点击两个拼图块交换位置
- 左上角眼睛图标：查看原图
- 右上角重置图标：重新开始本关
- 标题栏重置按钮：重新开始全部

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- LocalStorage 保存进度
- 响应式设计，支持手机/桌面

## 启动方式

```bash
cd puzzle-game
python3 -m http.server 8890
# 浏览器打开 http://localhost:8890
```
